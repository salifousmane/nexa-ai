# Nexa AI — Product Vision

## 1. Présentation

Nexa AI est une plateforme intelligente conçue pour aider les individus à comprendre les évolutions du monde, identifier les compétences nécessaires pour leur avenir et construire un parcours personnalisé d’adaptation.

Le monde professionnel et technologique évolue rapidement avec :
- l’intelligence artificielle ;
- l’automatisation ;
- la transformation numérique ;
- l’apparition de nouveaux métiers.

Beaucoup de personnes souhaitent évoluer mais ne savent pas :
- quelles compétences développer ;
- quelles directions prendre ;
- comment organiser leur progression.

Nexa AI répond à ce problème grâce à une intelligence artificielle personnalisée.

---

# 2. Mission

Permettre à chaque personne de mieux comprendre les changements du monde et de développer les compétences nécessaires pour évoluer.

---

# 3. Vision long terme

Créer un assistant intelligent d’évolution personnelle et professionnelle capable d’accompagner les individus tout au long de leur parcours.

Nexa AI ambitionne de devenir un compagnon permanent permettant de :

- détecter les changements importants ;
- recommander les compétences pertinentes ;
- créer des parcours personnalisés ;
- accompagner l’apprentissage ;
- mesurer la progression.

---

# 4. Problème

Les changements technologiques créent une nouvelle difficulté :

Le monde évolue plus vite que la capacité de nombreuses personnes à s’adapter.

Les utilisateurs rencontrent plusieurs obstacles :

- manque d’orientation ;
- surcharge d’informations ;
- absence de parcours personnalisé ;
- difficulté à identifier les compétences importantes ;
- manque d’accompagnement continu.

---

# 5. Utilisateurs

## Jeunes et étudiants

Problème :

"Quelles compétences dois-je apprendre pour mon avenir ?"

Besoins :

- orientation ;
- découverte des métiers ;
- plan d’apprentissage.

---

## Professionnels

Problème :

"Comment rester compétitif dans un monde qui change ?"

Besoins :

- évolution professionnelle ;
- nouvelles compétences ;
- adaptation aux technologies.

---

## Personnes éloignées du numérique

Problème :

"Comment comprendre les nouveaux outils ?"

Besoins :

- accompagnement simple ;
- apprentissage progressif.

---

# 6. Solution

Nexa AI fonctionne comme un conseiller intelligent.

Il réalise trois grandes missions :

## Comprendre

Analyser :
- profil ;
- situation ;
- objectifs ;
- compétences actuelles.

## Orienter

Identifier :
- compétences prioritaires ;
- opportunités d’évolution ;
- domaines importants.

## Accompagner

Créer :
- parcours personnalisé ;
- exercices ;
- conseils ;
- suivi de progression.

---

# 7. Différence avec un chatbot classique

Un chatbot répond à des questions.

Nexa AI accompagne une évolution.

La différence :

- mémoire utilisateur ;
- analyse du profil ;
- recommandations personnalisées ;
- parcours évolutif ;
- suivi dans le temps.

---

# 8. Principes du produit

Nexa AI doit être :

## Personnalisé
Chaque utilisateur possède un parcours différent.

## Accessible
Les explications doivent être adaptées au niveau de chacun.

## Pragmatique
L’objectif est de transformer les conseils en actions concrètes.

## Responsable
Nexa AI ne garantit pas un emploi ou un résultat automatique.

---

# 9. MVP OpenAI Build Week

La première version démontrera :

1. Création d’un profil utilisateur.
2. Diagnostic intelligent.
3. Identification des compétences prioritaires.
4. Génération d’un parcours personnalisé.
5. Assistant IA avec contexte utilisateur.
6. Tableau de progression.

---

# 10. Objectif final

Construire une IA qui ne répond pas seulement aux questions, mais qui aide les humains à évoluer dans un monde en transformation.
