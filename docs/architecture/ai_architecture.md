# Nexa AI — Architecture Intelligence Artificielle

## Rôle de l'IA

L'intelligence artificielle constitue le moteur de personnalisation.

---

## Modules IA

### Diagnostic

Analyse le profil utilisateur.

Entrées :
- situation ;
- objectifs ;
- compétences.

Sorties :
- forces ;
- lacunes ;
- recommandations.

---

### Générateur de parcours

Transforme le diagnostic en plan d'évolution.

---

### Mentor IA

Assistant conversationnel qui accompagne l'utilisateur.

---

## Prompts spécialisés

Le comportement IA est défini par :

- system_prompt ;
- diagnostic_prompt ;
- path_generator_prompt ;
- mentor_prompt.
